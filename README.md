# Startup Funding Analysis using SQL

## Project Overview

This project analyzes startup funding data using SQL to uncover funding trends, top-performing sectors, country-level investment patterns, and company funding distribution.

The goal is to demonstrate analytical SQL skills by answering business-oriented questions and generating actionable insights from startup funding data.

---

## Dataset

The dataset contains information about startups including:

- Company Name
- Category
- Country
- Funding Amount
- Funding Round
- Founded Year

---

## Business Questions Answered

1. Which startup categories receive the highest funding?
2. Which companies have raised the most funding?
3. How does funding vary across countries?
4. What are the funding trends over time?
5. What percentage of total funding is contributed by each category?
6. Which startups rank highest within their category?
7. How does funding change year-over-year?
8. Which categories dominate investment activity?

## Key Insights

### Funding Concentration

A small number of startup categories account for the majority of total investment, indicating investor preference toward specific industries.

### Funding Leaders

Top-funded startups significantly outperform the rest of the market, demonstrating strong capital concentration.

### Geographic Trends

Funding activity is concentrated in a few countries, highlighting regional startup ecosystem strengths.

### Yearly Funding Patterns

Investment levels fluctuate over time, suggesting the influence of market conditions and investor sentiment.
